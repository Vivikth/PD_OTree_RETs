For all categories, level 1 is the easiest level and level 4 is the hardest level.

Tabulation Tasks

Task Description: In each round there will be a grid filled with symbols. The task involves using mathematical abilities to tabulate the symbols.

Concealment Tasks

Task Description: In each round there will be a list of symbols. The task involves using analytical abilities to conceal each symbol.

Interpretation Tasks

Task Description: In each round there will be a single symbol. The task involves using visual abilities to interpret that symbol.

Replication Tasks

Task Description: In each round, there will be a row of symbols. This task involves using technology to reproduce each symbol.

Organisation Tasks

Task Description: In each round, there will be a sequence of symbols. This task involves using categorisation abilities to rearrange each sequence of symbols.
