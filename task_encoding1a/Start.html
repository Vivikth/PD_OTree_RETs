Example Encoding

The encoding table is given below (you may have to scroll horizontally to view it fully).

Consider encoding the string 'a0!d'. Using the table below, we can see that:

  • If the original character is 'a', then the encoded character is '?'.
  • If the original character is '0', then the encoded character is 'v'.
  • If the original character is '!', then the encoded character is 'g'.
  • If the original character is 'd', then the encoded character is '9'.

Combining all these in order, the string 'a0!d' would be encoded as '?vg9'.

Make sure you understand the above example before proceeding!
